# 双色球历史开奖数据

> 绝大多数数据从[baeshuai](https://www.cnblogs.com/baeshuai/p/18863450)下载，然后借助Claude Code补全最近数据，并生成了数据更新脚本。

## 数据

lottery_data.csv 是整理好的一份完整的双色球开奖数据。

CSV文件格式：

```
期号,开奖日期,红球,蓝球,排序红球,排序蓝球,出球顺序,销售额,奖池金额,星期
2025061,2025/6/1,03 07 09 12 18 23,05,03 07 09 12 18 23,05,23 18 12 09 07 03,365000000,2150000000,星期日
```

字段说明：

- 期号：格式为 YYYYNNN（年份+期数）
- 开奖日期：格式为 YYYY/M/D
- 红球：6个红球号码，空格分隔，两位数格式（如：01 02 03 04 17 22）
- 蓝球：1个蓝球号码，两位数格式
- 排序红球：红球号码按升序排列
- 排序蓝球：蓝球号码
- 出球顺序：实际出球顺序（可能不是升序）
- 销售额：单位为元
- 奖池金额：单位为元
- 星期：中文格式
