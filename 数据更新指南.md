# 双色球数据更新指南

## 快速更新（推荐）

### 方法一：使用自动更新脚本（最简单）

```bash
cd "D:\GitHub_Res\claude-双色球"
python simple_update.py
```

**说明：**
- 脚本会自动读取 `lottery_data.csv` 中最新的期号
- 从下一期开始尝试获取新数据（最多20期）
- 自动将新数据追加到CSV文件
- **自动按期号从新到旧排序**（最新期号在最前面）
- **每次运行前自动备份**原文件到 `lottery_data_before_sort.csv`
- 整个过程全自动，无需手动操作

## ✨ 新增功能：自动排序

### 排序规则
- **每次运行脚本后**，CSV文件会自动按期号从新到旧排序
- 最新期号在文件最前面（第2行，第1行是表头）
- 最早期号在文件最后面

### 自动备份
每次运行脚本时，会自动备份原文件：
- 备份文件名：`lottery_data_before_sort.csv`
- 位置：与CSV文件同一目录
- 用途：防止排序或更新出错时可以恢复

## 常见问题

### Q1: 脚本提示"期号不存在或未开奖"
**A:** 正常现象，说明已经是最新的数据，该期号还未开奖。

### Q2: 脚本运行失败怎么办？
**A:**
1. 检查网络连接
2. 确认网易彩票网站可访问：https://sports.163.com/caipiao/lottery/ssq/
3. 稍后再试（可能是网络波动）

### Q3: 如何验证数据是否正确更新？
**A:**
```bash
# 查看最后5条记录
cd "D:\GitHub_Res\claude-双色球"
tail -5 lottery_data.csv
```

### Q4: 更新前会自动备份吗？
**A:** `simple_update.py` 不会自动备份，建议手动备份：
```bash
cp lottery_data.csv lottery_data_backup_$(date +%Y%m%d).csv
```

## 数据备份建议

### 定期备份
```bash
# Windows PowerShell
cd "D:\GitHub_Res\claude-双色球"
Copy-Item lottery_data.csv "lottery_data_backup_$(Get-Date -Format 'yyyyMMdd').csv"
```

### 自动备份脚本
创建 `backup_and_update.bat`（Windows）：
```batch
@echo off
cd /d "D:\GitHub_Res\claude-双色球"
set BACKUP_FILE=lottery_data_backup_%date:~0,4%%date:~5,2%%date:~8,2%.csv
copy lottery_data.csv %BACKUP_FILE%
python simple_update.py
pause
```

使用方法：双击运行 `backup_and_update.bat`

## 注意事项

1. **网络要求**：需要能够访问网易彩票网站
2. **文件格式**：CSV文件必须使用UTF-8编码
3. **数据完整性**：脚本会自动跳过已存在的期号
4. **错误处理**：如果某期获取失败，会继续尝试下一期
